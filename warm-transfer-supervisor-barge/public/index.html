Warm Transfer - Supervisor Barge

Incoming calls are bridged to the agent destination below. Use the controls to attach a supervisor to any active call.

! Setup & Usage Instructions

Installation

  1. Clone the repository and navigate to the warm_transfer directory.
  2. Install dependencies: npm install
  3. Create a .env file with the following variables:
    • TELNYX_API_KEY — Your Telnyx API key
    • TELNYX_CONNECTION_ID — Your Call Control connection ID
    • TELNYX_PHONE_NUMBER — Your Telnyx phone number (E.164 format)
    • AGENT_PHONE_NUMBER — Default agent destination (phone or SIP URI)
    • PORT — Server port (defaults to 3000)
  4. Ensure SSL certificates exist at the configured path, or update index.js for your environment.
  5. Start the server: npm start (or npm run dev for auto-reload)

How to Use

  1. Set the webhook URL on your Telnyx Call Control connection to https://<your-domain>/webhooks.
  2. Update the Agent Destination below to the phone number or SIP URI where incoming calls should be forwarded.
  3. When a call comes in, it will appear under Active Calls and automatically bridge to the agent.
  4. Once bridged, enter a supervisor phone number or SIP URI and select a role:
    • Barge — Supervisor can speak to both parties
    • Whisper — Supervisor can speak only to the agent
    • Monitor — Supervisor listens silently
  5. Click Attach Supervisor to dial the supervisor into the call.
